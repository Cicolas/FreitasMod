# FreitasMod
Freitas mod is a terraria mod

---

Freitas mod is a mod that add to the game more events, bosses, NPCs, weapons and more...

new Boss - Neymar Messi Boss (he's Neymar, and he's Messi, but also is Neymar, and...)

new NPC - Freitas (Do you need buy some F of Freitas?)

new Staff - Freitasss staff (who never though of having Freitasss at your service?)

2 new Guns - MiniFreitasss (it's like mini shark, but with Freitasss, much better isn't?)
			 MegaFreitasss (it's the MiniFreitasss for true heroes)

new Ammo - Freitasss (Freitas Super Small)

new Throwable weapon - Freitasss Ball (yes Starbound steal our idea)

new Event - Freitas Day (a day with very, very much R6!!!)

4 new Decoratives Blocks - Gold F Statue
						   Platinum F statue
						   Celestial F statue
						   Freitas statue
